# RealEstate-Project-25
This project is a Real Estate Management System developed as part of the Computer Engineering course for the Fall 2025-2026 semester. The project was managed using the Kanban methodology and designed in accordance with object-oriented programming principles.

It is a platform that allows users to list and filter properties for sale or rent and interact with real estate agents.
